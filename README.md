# Online Voting System

A web-based online voting system developed using Python and Flask.

## Features
- Voter authentication
- Secure vote casting
- Prevents duplicate voting
- Stores votes using SQLite database
- Simple and user-friendly interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
